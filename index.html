Mashify

Mashify is a React.js app allows users to create Spotify playlists featuring artists similar to two chosen artists, allowing them to discover new music. This project makes use of Spotify's Web API.

ArtVista

ArtVista is a collaboratively made, virtual gallery, allowing users to view, post and comment art. This project was created with React.js and Node.js and uses Google Cloud Platform to store artworks.

Music of Life

Music of Life uses JavaScript to provide a musical representation of the Game of Life, combining my passion for music and maths. Users explore and listen to different patterns.

TrackIt

TrackIt is a personal financial tracker. Developed with a React.js frontend and Node.js backend, users can login and create transactions to log their spending.
